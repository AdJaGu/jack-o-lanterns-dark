# jack-o-lanterns-dark
Jack-o'-lanterns (pumpkins) Firefox Quantum (57.0) theme with a dark background and light text.

## Why?
I wanted a dark theme for Firefox Quantum (57.0) that did not have a grey tinted toolbar.
